# rtu
RTU (mvts3g) API client

* SOAP client (template from https://github.com/hooklift/gowsdl)
* Query builder
* Simple string filter (Where("in_ani <> 123123123") for example)
* Advance filter from map (Filters("or", []string{"condition1", "condition2"}))
* Get CDRs
